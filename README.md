# Vista Hackathon

## Colab Setup
To get started on Google Colab, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Woeter69/vista.git
   cd vista
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Download the dataset file from Google Drive:
   ```bash
   python download_data.py "https://drive.google.com/file/d/YOUR_FILE_ID/view?usp=sharing" --output dataset.zip
   ```
4. Unzip the dataset:
   ```bash
   unzip dataset.zip -d data/
   ```
